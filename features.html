Activities Configurator Extension Features

  • The Gnome Shell Extension Preferences tool can be executed by a secondary mouse click on the Activities Text or Icon.
  • An icon can be selected to precede the text.
  • The icon can be hidden.
  • The icon padding is adjustable.
  • The Activities Text can be replaced by text chosen by the user. Pango formatting can be applied to the text. See Pango Markup Format.
  • The text can be hidden.
  • The text padding is adjustable
  • The Activities Button can be removed.
  • The Hot Corner Threshold can be adjusted.
  • The Hot Corner can be disabled.
  • The Panel's rounded corners can be hidden.
  • The application menu button can be hidden.
  • The Shell Theme can be optionally overridden.
  • The Overview can be displayed if no applications are running (at login and whenever the last application window is closed).
  • The Panel Background can be changed.
  • The Panel background color transparency is adjustable.
  • A configurable Shadow Effect can be added to the Panel.
  • Setting the Shadow's Transparency, Vertical Length, Blur Radius, and Spread Radius determines the shadow's appearance.
  • The appearance of the Panel's background can be chosen when a window is maximized.
  • The Activities Button can be moved to the right corner of the Panel.
  • Conflicts with other extensions can be resolved programmatically in most cases.
