Installation from the Gnome Shell Extension Website

Installation is simple, just slide the switch to ON.

You will be prompted to Cancel or Install. Select Install..


After installation the website can be used to enable, disable, set extension's preferences,
or remove the extension.


Alternative Installation from the Gnome Shell Extension Website

Gnome Shell Extensions can be installed from an extension zip file with the gnome-tweak-tool.  

  • Download extension zip file from the extension website zip files.
     
  • Install the zip file with the Tweak Tool.
     
    Click (None) then select and open the zip file.
     
  • Log out and then log in to start a new Gnome Shell session.
  • Use the Tweak Tool or the website to enable/disable and set preferences.
