Installation from the Gnome Shell Extension Website

Gnome Shell Extensions should be installed from the extensions.gnome.org website.   All extensions available on the ego website have been reviewed before their acceptance for distribution.   The review process makes it very unlikely an extension downloaded from the website contains malicious code.   The website will determine the latest version of the extension available for your version of the Gnome Shell.   The website requires the browser Gnome Shell Integration plugin.   Mozilla Firefox comes with the plugin installed.


Installation is simple, just turn the switch to ON.


After installation the website can be used to set extension's preferences and enable or disable the extension.

Alternative Installation from the Gnome Shell Extension Website

Gnome Shell Extensions can be installed from an extension zip file with the gnome-tweak-tool (Tweak Tool).   

  • Download extension zip file from the extension zip file page.
  • Install the zip file with the Tweak Tool. 
    Click (None) then select and open the zip file.
  • Log out and then log in to start a new Gnome Shell session.
  • Use the Tweak Tool to enable extension and set preferences.
