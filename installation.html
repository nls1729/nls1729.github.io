Installation from the Gnome Shell Extension Website

Installation is simple, just slide the switch to ON.

You will be prompted to Cancel or Install. Select Install..

After installation the website can be used to enable, disable, set extension's preferences,
or remove the extension.

Alternative Installation from the Gnome Shell Extension Website

Download extension zip file from the extension website zip files.

Gnome Shell Extensions can be installed from an extension zip file with the
gnome-tweak-tool for versions of the Gnome Shell prior to 3.26.

  • Install the zip file with the Tweak Tool.
     
    Click (None) then select and open the zip file.

For installation in Gnome Shell version 3.26 and later, the zip file can be installed
manually from a command line prompt ($). An example for zip file
activities-config@nls1729.v58.shell-extension.zip follows.

  • Determine the UUID of the extension from the extension's zip file name.
    activities-config@nls1729
  • Set variables.
    $ ZIPFILE=~/Downloads/activities-config@nls1729.v58.shell-extension.zip
    $ EXTDIR=~/.local/share/gnome-shell/extensions/activities-config@nls1729
  • Remove existing extension if it exists.
    $ rm -rf "${EXTDIR}"
  • Create extension directory.
    $ mkdir "${EXTDIR}"
  • Unzip zip file into extension directory.
    $ unzip "${ZIPFILE}" -d "${EXTDIR}"

To complete the installation.

  • Log out and then log in to start a new Gnome Shell session.
  • Use the Tweak Tool or the website to enable/disable and set preferences.
