The preferred method of installation is to use the extension's page on the
Gnome Shell Extension Website. Sometimes re-installing an earlier version
is useful for trouble shooting or other reasons.

The zip files are are hosted on the extensons.gnome.org wedsite.
The Gnome Shell Integration plugin is not required.
Download the version you need based on the Gnome Shell version installed.

On the index page use the secondary mouse button (right for right handed or
left for left handed) to click the (v)ersion number then select open new tab or
window to download the extension zip file.
